# Reel Generator Commands

Here are all the available commands to generate and publish your reels. 

## Batch / Script Commands

### 1. Single Reel Generation
Generate a reel for a specific sign. Prompts for input if the sign is not provided.
```cmd
generate_single.bat
generate_single.bat Aries
```

### 2. All Signs Generation
Generate reels for all 12 signs at once. Defaults to `horoscope` category. Generates in parallel.
```cmd
generate_all.bat
generate_all.bat relationships
generate_all.bat career
generate_all.bat current_events
generate_all.bat emotional_healing
generate_all.bat manifestation
generate_all.bat all
```

### 3. Viral Reels Generation
Generate viral reels for a specific sign and category.
```cmd
generate_viral.bat
generate_viral.bat Leo relationships
generate_viral.bat Scorpio manifestation
generate_viral.bat Aries emotional_healing
generate_viral.bat all career
generate_viral.bat all all
```

### 4. Publish Queue Workflow
Generate reels for all signs and create same-name JSON queue files for auto-posting.
```cmd
generate_publish_queue.bat
generate_publish_queue.bat horoscope
generate_publish_queue.bat relationships
```

## Direct Python Usage

### Generate Reels (`Generate.py`)
```cmd
python Generate.py --sign Pisces --category current_events
python Generate.py --sign all --category all --parallel
python Generate.py --sign Leo --soundtrack "soundtrack/song.mp3"
python Generate.py --sign all --category horoscope --parallel
```

### Generate Publish Queue (`generate_publish_queue.py`)
```cmd
python generate_publish_queue.py --sign all --category horoscope --parallel
python generate_publish_queue.py --sign Aries --category relationships
```

### Post Scheduled Reels (`post_scheduled_reels.py`)
Post due reels from the publish queue (typically used in automated environments like GitHub Actions).
```cmd
python post_scheduled_reels.py
python post_scheduled_reels.py --queue-dir publish_queue

## Fun Genre Reels

Three new viral content types that go beyond per-sign horoscopes.
Each uses a **different layout** — hero badge (giant letter / month / sign pair) + insight cards.

| Genre | ID | Subjects | Reels |
|---|---|---|---|
| Name Initials | `name_initials` | 13 letter groups (A&J, B&K … Y&Z) | 13 |
| Lucky Month | `lucky_month` | 6 month pairs (Jan&Feb … Nov&Dec) | 6 |
| Compatibility | `compatibility` | 66 unique zodiac pairs | 66 |

### Generate with `Generate.py`

```cmd
# Name initials — one group
python Generate.py --genre name_initials --subject AJ
python Generate.py --genre name_initials --subject ST

# Name initials — all 13 groups
python Generate.py --genre name_initials --subject all

# Lucky month — one pair
python Generate.py --genre lucky_month --subject MAY_JUN
python Generate.py --genre lucky_month --subject JAN_FEB

# Lucky month — all 6 pairs
python Generate.py --genre lucky_month --subject all

# Compatibility — one pair (use first-3-letters of each sign, uppercase)
python Generate.py --genre compatibility --subject ARI_SCO
python Generate.py --genre compatibility --subject LEO_LIB

# Compatibility — all 66 pairs (use --parallel to speed up)
python Generate.py --genre compatibility --subject all --parallel

# All genres at once
python Generate.py --genre all --subject all --parallel
```

### Add to Publish Queue

```cmd
# Queue all name initials reels
python generate_publish_queue.py --genre name_initials --subject all

# Queue all lucky month reels
python generate_publish_queue.py --genre lucky_month --subject all

# Queue a single compatibility pair
python generate_publish_queue.py --genre compatibility --subject ARI_SCO

# Queue all 66 compatibility reels (parallel generation + queued for posting)
python generate_publish_queue.py --genre compatibility --subject all --parallel

# Skip generation, only queue from existing output files
python generate_publish_queue.py --genre name_initials --subject all --skip-generate
```

### Subject Slug Reference

**`name_initials`** slugs:
`AJ` `BK` `CL` `DM` `EN` `FO` `GP` `HQ` `IR` `ST` `UV` `WX` `YZ`

**`lucky_month`** slugs:
`JAN_FEB` `MAR_APR` `MAY_JUN` `JUL_AUG` `SEP_OCT` `NOV_DEC`

**`compatibility`** slugs — first 3 letters of each sign, uppercase (e.g. `ARI_SCO`, `LEO_LIB`, `GEM_CAP`)

Full list of compatibility slugs can be generated by listing `fun_genre_categories.COMPATIBILITY_PAIRS`.
