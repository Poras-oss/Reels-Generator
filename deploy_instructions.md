# 🚀 Deploy Instructions (Deprecated)

> **⚠️ This file is outdated.** The project now runs locally on Windows.
>
> Please see **[WINDOWS_SETUP_AND_WORKFLOW.md](WINDOWS_SETUP_AND_WORKFLOW.md)** for the
> complete setup guide, daily workflow, and Instagram posting strategy.

The old cloud-based deployment (Linux VPS + cron + Meta Graph API) has been
replaced with a simple local workflow:

1. Double-click `setup_windows.bat` (one-time)
2. Double-click `generate_all.bat` (daily)
3. Upload reels from `reels_output/` to Instagram manually

See `WINDOWS_SETUP_AND_WORKFLOW.md` for full details.
